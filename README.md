# DSPIC OQPSK，PI/4 SHIFT QPSK変調器
## はじめに
2.4GHz用の無線機に使うコントローラのテストプログラム用リポジトリです．
このプログラムには，無線機ICのコントローラはサポートしていないです．
最終的には無線機のリポジトリから参照するようにさせます．

## 仕様
- マイコン<br>
    DSPIC33CH64MP202

- 使用モジュール
  - DMA<br>
      最大3個（変調方式による）
  - PWM<br>
      1個

  - フラッシュ<br>
      LUTの格納様に10KByte程度（sinデータテーブルのサイズによる）
      本体プログラム分

  - メモリ<br>
      LUTと，送信データバッファのサイズによる

## 動作
このプログラムはcpuを使わないで変調した信号を出力するために作成しました．
LUTにあらかじめ計算したデータを格納しておきDMAで高速に転送することでCPUを使用しない様にします．
